# Time Series Timeline

A simple web application to enable using VSTS for repoting activities on projects and organizations.
![/docs/timeline.png](/docs/timeline.png)

# Install

Dev:
```sh
npm run install:dev
```

Production:
```sh
npm install
```

# Run

Dev:
```sh
npm start:dev
```

Production:
```sh
npm start
```

# Build

```sh
npm build
```
